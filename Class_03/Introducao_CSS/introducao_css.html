Learning Css Style

 Css style funciona dentro do componenete Html, recomenda-se usar apenas até três styles dentro de um elemento.
 Você pode optar posso esse tipo de css quando quer arrumar alguém detalhe trocar apenas uma cor, sempre usado para coisas simples,
confira abaixo exemplos de como usar o Css style:

Hello World

Se você clicar no botão acima ira aumentar a fonta da palavra Hello World usando style, teste.

Hello World

Se você clicar no botão acima ira mudar a cor da palavra Hello World usando style, teste.

Hello World
